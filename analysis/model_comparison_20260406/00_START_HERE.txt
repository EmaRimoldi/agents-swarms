─────────────────────────────────────────────────────────────────────────────

In swarm learning, SPEED > RAW CAPABILITY

Haiku (smallest, fastest) outperformed Opus (largest, slowest) because:
1. Faster inference → more iterations in fixed time
2. More iterations → better solution exploration
3. Swarm collaboration compensates for individual model size

This challenges the common assumption that "larger models = better results"
